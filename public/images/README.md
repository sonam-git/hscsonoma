## Image Guidelines

1. **Format**: Use WebP or JPEG for photos, PNG for logos/graphics with transparency
2. **Optimization**: Compress images before uploading (use tools like TinyPNG or Squoosh)
3. **Aspect Ratios**: 
   - Hero images: 16:9 or 21:9
   - Card thumbnails: 4:3 or 16:9
   - Profile photos: 1:1 (square)
4. **Alt Text**: Always include descriptive alt text in the code
5. **File Size**: Keep images under 200KB when possible for fast loading
